Question: Solve the integer program with n=8 using a cutting plane algorithm, adding violated triangle inequalities as cutting planes. The objective function coefficients c are as follows:


What is the optimal ordering? (Hint: the optimal value is -22.)

You can download an AMPL model file and a data file for this problem. Some hints about using AMPL are now available, including methods for adding constraints to this linear ordering problem.


Solution: Here is the output from an AMPL run. Taking the original model file and data file leads to a solution with value -35. This violates the constraints listed here. Adding the cuts leads to a solution with value -25, which violates the cuts listed here. The addition of these cuts results in the optimal solution to the LP relaxation being the incidence vector of an ordering. The optimal ordering is 6 - 3 - 1 - 4 - 5 - 8 - 2 - 7.
