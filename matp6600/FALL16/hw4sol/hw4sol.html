  2. Use the first order optimality conditions to find the global minimizer for the problem

    (Note that this problem is nonconvex — prove that the solution you obtain is the global minimizer, without considering the second order conditions.)

    Solution: The constraints force x1 ≥ 1 and x2 ≥ 3, so x1x2 →∞ as ||x||→∞ for feasible x. Thus, a global minimizer exists. The constraints are linear, so constraint qualification holds. Thus, any local minimizer is a KKT point, so it suffices to find the best KKT point. The KKT conditions can be written:

    We break into cases exploiting the complementarity conditions (12)–(14).
    • Assume u1 > 0: From (12), we have x1 = 1, so by (11) either u2 > 0 and/or u3 > 0. If u2 > 0 then from (13) we have x2 = 3 but then (20) is violated. So the only option is u3 = 1. Then from (14) we have x2 = 9. We then get the KKT point x = (1,9) with value 9 by setting u1 = 6.
    • Assume u1 = 0: From (19) and (10), we have u3 = x2 ≥ 1, so from (14) we must have 3x1 + x2 = 12. We break into subcases based on (13):
      • Assume u2 > 0: Then we get the KKT point x = (3,3) with value 9 by setting u3 = 1 and u2 = 2.
      • Assume u2 = 0: Then we have x1 = u3, x2 = 3u3, and 3x1+x2 = 12. Solving these equations gives the KKT point x = (2,6) with value 12 by setting u3 = 2.

    Thus, we have two global minimizers, each with value 9: x = (1,9) and x = (3,3).
