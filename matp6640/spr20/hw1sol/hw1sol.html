  4. Show that a convex cone has at most one extreme point, namely the origin. Give an example of a convex cone with no extreme points.

    Solution:

    Assume x≠0 is an extreme point of the cone, which we designate as K. Then x1 := 0.5x and x2 := 1.5x are two distinct points in K, since it is a cone. We have x = 0.5x1 + 0.5x2 so x is not an extreme point. So the only possible extreme point is the origin.

    Any subspace is a convex cone without an extreme point. So for example we can take ℝn for any finite positive n.
