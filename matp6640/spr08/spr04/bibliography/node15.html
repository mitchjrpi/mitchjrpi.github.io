Miscellaneous

An interior point approach to multicriteria optimization: Fliege [12].

Given an optimal solution, how do you pick the data so that this is optimal? Variants of this inverse optimization problem are explored in Ahuja and Orlin [1].

An example of a problem which can be formulated as an LP, but for which other more efficient approaches exist, is discussed by Osborne [34].
