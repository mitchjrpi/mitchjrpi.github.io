Network flows

The simplex algorithm has been specialized to solve network flow problems. It is necessary to make some adjustments when using an interior point method for solving these linear programs for the algorithm to be efficient. Resende et al. [34,44] have described the appropriate changes. They've also [43] discussed the identification of the optimal basis.
