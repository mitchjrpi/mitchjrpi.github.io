Column generation theory

Some other optimization problems can be solved by solving a sequence of linear programming problems. The branch and bound approach and the cutting plane approach to solving integer programming problems have this framework. There is a fair amount of active current research on interior point column generation methods. Atkinson and Vaidya [10] gave the first polynomial interior point column generation method; this has been extended to a long-step method by Ramaswamy and me [35]. Goffin and Vial [19] showed that multiple cuts can be added at once while retaining a pseudopolynomial algorithm; these same authors [20] surveyed the area. Gondzio [21] discussed practical warm start methods, and Yildirim and Wright [54] also discuss warm start methods.
