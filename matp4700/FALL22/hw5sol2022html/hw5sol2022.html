  • 6.6.24 Pivots consist of pushing flow around cycles. If the current flow is integral and if the bounds on the flows on each edge are integral, then the maximum additional flow that can be pushed around the cycle is integral. Thus, the flow remains integral after each pivot. If a linear program has an optimal solution then the simplex algorithm will find that solution, so in the case of a transportation problem, the optimal solution x* returned by the algorithm must be integral.
