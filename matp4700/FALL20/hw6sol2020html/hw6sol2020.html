  • 6.6.21

    (a) Only basic variables, plus the incoming arc. Must constitute a cycle in the graph. Have either 0 or 2 entries in each row and each column.

    (b) Depth first search: pick a basic edge that shares an endpoint with the incoming edge, then pick a basic edge that shares an endpoint with the edge just picked. Backtrack if no such edge exists. Proceed until return to the other endpoint of the incoming edge.

    (c) Given a spanning tree, a single additional edge creates a unique cycle.
