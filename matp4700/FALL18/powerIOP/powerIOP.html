1 Selling power externally

ZICO has the option to sell power to a neighboring county. The selling price is $25 per MWh. In order to sell the power, ZICO has to rent a transmission line, which costs $100 for each 6-hour period. The transmission line has a capacity of 4MW, which means it can transmit up to 24 MWh each 6 hours. How much should ZICO sell in each 6 hour period? How much steam power should they generate in each 6 hour period?

Now consider a daily rental model, where the transmission line is rented by the day for $300 per day. Which days should they rent the line? How much should ZICO sell in each 6 hour period? How much steam power should they generate in each 6 hour period?

If you prefer to use them, my model and data file are available on LMS and the course website. The solution to the LP relaxation has value 9309.52381. If your model returns a different value, or if your model is nonlinear, you need to either modify your model so it does give this value, or use my model.

2 AMPL notes

  • Variables can be defined to be binary in AMPL.
  • You may find it useful to use the floor function to tie together decisions for a day and decisions for a six hour period. It is permitted in AMPL to use this function within the argument of a variable. For example, if you have a variable z{1..horizon/4} and you have an index t in {1..horizon}, you can define a constraint that involves z[1+floor((t-1)/4)].
