Solution:

(a)
Note that x11 > 0, x 22 > 0 and x 31 > 0. So dual constraints 1, 2, and 3 must all be satisfied at equality by any dual optimal solution.

4.
(20 points) Let A be an m × n matrix. We consider the system

(I) There exists x ≥ 0 with Ax ≤ 0 and x≠0.

(a)
Write down a linear program (P) with nonnegative variables x ≥ 0 and m constraints Ax ≤ 0, which can be used to determine whether System (I) has a solution.
(b)
Show that (P) has a feasible solution.
(c)
What is the dual (D) to your linear program (P)?
(d)
What are the possible cases for (P), (D), and (I)? (Note: System (I) is either consistent or inconsistent. (P) and (D) are either infeasible, feasible with a finite optimal value, or feasible with an unbounded optimal value. You are required to enumerate the possible cases for the combination of the three.)

Solution:

(a)
In order to force x to have a nonzero component, we can maximize eT x, where e is the vector of ones.
(b)
x = 0 is feasible in (P).
(c)
The dual is given on the front page.
(d)
There are only 2 cases:
  • (I) is consistent ⇐⇒ (P) has a feasible solution with positive objective function value ⇐⇒ (P) has an unbounded optimal value, since we can scale any feasible solution with positive value ⇐⇒ (D) is infeasible.

    Note that we can’t have both (P) and (D) infeasible, from part (b).

  • (I) is inconsistent ⇐⇒ (P) has no feasible solution with positive objective function value ⇐⇒ (P) has optimal value 0, since we know it is feasible from (b) ⇐⇒ (D) also has optimal value 0.

(a)
(5 points) Find a dual solution that satisfies complementary slackness.
(b)
(10 points) Find a better basic feasible solution.

Solution:

(a)
We set u1 = 0 and use the chain reaction, exploiting ui + vj = cij for basic arcs.
(b)
The arc (2, 2) has c22 - u2 - v2 = -1, so this arc enters the basis. It created a cycle s2 → d2 → s1 → d4 → s2. We can push up to 10 units around this cycle, at which point the flow on 2 of the arcs drops to 0. We arbitrarily pick one of them to leave the basis.
