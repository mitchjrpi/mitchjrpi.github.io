  • 4.6.42 The change in the objective function value at each iteration is the product of the cost cj with the minimum ratio for the pivot column. If the linear program is nondegenerate then the value of the minimum ratio is always nonzero, so there is a strict improvement in the objective function value at every iteration. Thus, we can never return to an earlier tableau, so we cannot cycle.
